# SE_Pruning
This project is designed to perform pruning on the optimal performance networks for deep learning-based speech enhancement solutions, aiming to significantly reduce the computational load and storage requirements of the networks while maintaining enhancement performance.

The base code is referenced from https://github.com/Dahan-Wang/Rethinking-Flow-and-Diffusion-Bridge-Models-for-Speech-Enhancement.git, and the pruning work in this project is built upon this codebase.
